alternative AdBlock (Plus) Whitelist

Diese Whitelist soll eine Alternative zu der acceptable adds Whitelist von AdBlock Plus darstellen. Im Gegensatz zur original Whitelist verfolge ich keinerlei kommerzielle Interessen mit der Erstellung der Liste. Es gibt somit auch keine Müglichkeit, sich auf der Liste "einzukaufen". Die Liste verfolgt einzig und allein den Zweck, einen Kompromiss zwischen den monetarisierungsinteressen von Webseitenbetreibern und dem Wunsch der Webnutzer, von nerviger Werbung verschont zu werden, darzustellen. Aus diesem Grund enthält die Liste auch keinerlei Filter, welche Flash-Banner durchlassen.

Die Liste kann über diesen Link abonniert werden.

Spezial-Listen

Die Spezial-Listen enthalten jeweils nur eine Teilmenge der Whitelist und sind für Personen gedacht, die nur auf bestimmten Seiten-Typen Werbung zulassen möchten.

Zur Zeit gibt es die Spezial-Listen für folgende Seiten-Typen:

  • Blogs* (abonnieren)
  • News-Seiten* (abonnieren)
  • sonstige Seiten* (abonnieren)
  • gemeinsame Filter (abonnieren)

Die mit * gekennzeichneten Listen benötigen die Liste der gemeinsamen Filter, um einwandfrei zu funktionieren.

Durch einen Kick auf den Seiten-Typ öffnen Sie die jeweilige Spezial-Liste zur Ansicht.

Social-Media

Speziell für soziale Netzwerke und andere Social-Media-Seiten gibt es eine seperate Whitelist, welche Werbung auf diesen Seiten erlaubt. Die Einträge auf dieser Liste sind nicht Bestandteil der normalen Whitelist!

Die Liste für Social-Media Seiten kann über diesen Link abonniert werden.

Mitmachen

Änderungen an der Liste können über das git Repository auf GitHub verfolgt werden. Dort können auch Vorschläge für weitere Ausnahmen eingereicht werden.
