Diese Whitelist soll eine Alternative zu der acceptable adds Whitelist von AdBlock Plus darstellen.

Im Gegensatz zur original Whitelist verfolge ich keinerlei kommerzielle Interessen mit der Erstellung der Liste. Es gibt somit auch keine Müglichkeit, sich auf der Liste "einzukaufen".

Die Liste verfolgt einzig und allein den Zweck, einen Kompromiss zwischen den monetarisierungsinteressen von Webseitenbetreibern und dem Wunsch der Webnutzer, von nerviger Werbung verschont zu werden, darzustellen.

Aus diesem Grund enthält die Liste auch keinerlei Filter, welche Flash-Banner durchlässt.

Die Liste kann über diesen Link abonniert werden.

Die Liste für Social-Media Seiten kann über diesen Link abonniert werden.

Änderungen an der Liste können über das git Repository auf GitHub verfolgt werden.

Vorschläge für zusätzliche Filter können hier eingereicht werden.
